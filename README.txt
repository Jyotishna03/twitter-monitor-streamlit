# 🚨 Twitter Sentiment Monitor

A real-time tweet monitoring web app built with **Streamlit**, leveraging **Twitter scraping** and **sentiment analysis** to detect and alert on negative tweets instantly via **Telegram**.

---

## 🌐 Live Demo
https://twitter-monitor-app-acbggs9xmetkbz9qvbc7xu.streamlit.app/

---

## 📌 Features

- 🧠 **Sentiment Analysis** with `TextBlob`
- 📲 **Instant Alerts via Telegram Bot**
- 💡 **Light/Dark Theme Toggle**
- 📥 **Downloadable Sentiment Report (CSV)**
- 🖥️ **Custom UI with Streamlit**
